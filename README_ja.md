# 都市水循環システム (Urban Eco-Pulsing Water Infrastructure: UEPWI)

> 英語版はこちら: [README.md](./README.md)

> 本リポジトリは概念的提案および理論的枠組みを提示するものであり、UEPWIが実証済みのシステムであること、またヒートアイランド、粉塵、花粉問題を確実に解決できることを主張するものではありません。実装には、科学的モデリング、実証実験、公衆衛生評価、インフラ適合性評価、ガバナンス設計が必要です。

## 概要

**都市水循環システム (UEPWI)** は、「自然補完科学」の哲学に基づいた概念的フレームワークです。都市に局所的な水循環を回復させること、すなわち都市の「代謝」機能を再構築することが、相互に関連する複数の都市環境課題に対する統合的かつ多角的なアプローチになり得るという仮説を探求しています。

この構想の核心は、都市を単なる乾燥して熱を溜め込む人工構造物としてではなく、制御された水循環によって部分的に「呼吸し、汗をかく」機能を取り戻し得る都市システムとして捉え直すことにあります。

## CO₂削減に関するポジション

UEPWIはCO₂削減の代替ではありません。CO₂削減は引き続き必要であり、UEPWIは都市環境における補完的な適応策・緩和支援策として位置づけられます。地球規模の排出量削減が重要な長期目標である一方で、都市には深刻な環境ストレスを管理するための即時的かつ局所的な適応戦略も必要です。

## 1. 背景

現代の都市環境は、アスファルトやコンクリートなどの広範な不浸透性表面と、雨水を速やかに排出する排水システムによって特徴付けられています。このインフラ設計は、水の浸透と蒸発という自然のプロセスを大きく減少させ、都市の水循環を断絶させてきました。

その結果、都市は複合的な環境ストレスに直面しています。

- **ヒートアイランド現象 (UHI)**: 熱を吸収する表面が、周囲の気温や表面温度を上昇させます。
- **乾燥した表面と粉塵**: 表面の水分が不足しているため、PM2.5やPM10などの粒子状物質が風や交通によって大気中に再飛散しやすくなります。
- **空気中のアレルゲン**: 花粉は、乾燥して高温の都市空気中では、より遠くまで飛散し、長く滞留する可能性があります。
- **雨水の喪失**: 大量の雨水が急速に排水されることは、水資源の喪失であると同時に、下流域の洪水リスクを高める要因にもなります。

## 2. オリジナルのコンセプト

UEPWIの核心は、都市を「乾燥して熱を溜め込む構造物」から、制御された水循環によって「汗をかき、呼吸する」ことができる環境へと変えることです。

これには以下が含まれます。

- **地下雨水タンク**: 雨水の表面流出を捕捉し、大規模な地下貯水槽に貯留します。
- **自動散水・ミストシステム**: 道路、歩道、中央分離帯、街路樹、緑地帯に、ターゲット型のミストおよび灌漑システムを展開します。
- **ソーラーポンプ**: 太陽光発電などの再生可能エネルギーを利用して水の分配を行い、運用時の排出量削減を目指します。
- **水質濾過**: エアロゾル化された水や公共空間で分配される水の安全性を確保するために、厳密な濾過・殺菌システムを実装します。
- **統合的な水源確保**: 収集した雨水、安全に処理された生活排水（中水）、および生態系への影響を考慮して慎重に管理された河川水の取水を、必要に応じて組み合わせます。

## 3. 提案されるメカニズム

提案されているシステムは、以下の作用メカニズムを仮説としています。

- **気化熱冷却**: 水が液体から気体へ相変化する際に潜熱を吸収し、周囲の空気を冷却する可能性があります。
- **表面温度の低下**: 熱せられたアスファルトやコンクリートに水を散布することで、表面温度とその後の熱放射を低減できる可能性があります。
- **粉塵再飛散の抑制**: 表面を湿らせることで、粉塵粒子と地面との凝集力が高まり、空気中への再飛散を抑えられる可能性があります。
- **花粉の捕捉**: エアロゾル化された水滴が、空気中の花粉を物理的に捕捉し、地面に洗い流す可能性があります。
- **植生の蒸散支援**: 都市の緑地インフラへ継続的に給水することで植物の健康を改善し、自然の遮光と生物学的蒸散を強化できる可能性があります。
- **冷房エネルギー需要の削減**: 都市の気温を下げることで、冷房度日（CDD）とエアコンに必要なエネルギーを理論的に削減できる可能性があります。
- **洪水緩和**: 雨水を発生源で捕捉することで、都市の排水システムへのピーク負荷を軽減できる可能性があります。

## 4. 季節ごとの考慮事項

UEPWIは、季節ごとの環境ストレスに適応する動的なシステムとして構想されています。

- **夏**: 高頻度のミスト散布と表面冷却によるヒートアイランド現象の緩和に主眼を置きます。
- **春**: 花粉の飛散が多い時間帯に、ターゲットを絞ったミスト散布による花粉抑制に主眼を置きます。
- **乾季**: 粉塵の制御と、必要に応じた限定的な表面水分の維持に主眼を置きます。
- **冬 / 凍結リスク**: 配管の破裂や危険な路面凍結を防ぐため、厳密な温度ベースのシャットオフと自動パージ（水抜き）プロトコルが必要です。
- **乾燥・インフルエンザに関する仮説**: 冬季の局所的な湿度を上げることで、インフルエンザなど一部の空気感染ウイルスの生存率を低下させる「かもしれない」という理論的仮説があります。ただし、これはあくまで仮説であり、実証された医学的主張ではありません。

## 5. システムアーキテクチャ

完全なUEPWIの展開には、理論上以下の要素が必要となります。

- **集水**: 集水域、雨水管、生活排水（中水）の分流システム。
- **貯留**: オーバーフロー管理機能を備えた地下貯水槽。
- **濾過**: 病原体の伝播を防ぐための多段階濾過・殺菌。物理濾過、UV処理、塩素消毒などが候補になります。
- **分配**: 低流量パイプ、ミストノズル、点滴灌漑ラインのネットワーク。
- **センサー**: 温度、湿度、土壌水分、PMレベル、必要に応じて花粉量を測定する環境モニタリングネットワーク。
- **制御アルゴリズム**: リアルタイムのセンサーデータと天気予報に基づいて散水スケジュールを最適化する、AI駆動またはルールベースのシステム。
- **再生可能エネルギー**: 実現可能な範囲で統合された太陽光発電アレイとバッテリー蓄電。
- **維持管理**: 定期的なフィルター交換、センサーのキャリブレーション、ノズル清掃、配管洗浄、季節ごとの停止手順。

## 6. 安全性と制限事項の要約

UEPWIの実装には重大な課題が伴います。主な制限事項としては、濾過が不十分な場合のレジオネラ菌などの水系病原体のリスク、すでに高温多湿な環境を過剰に加湿することによる体感温度悪化の可能性、高い初期費用と維持管理コスト、複雑な水利権のガバナンス、そして過密な都市部への後付け（レトロフィット）という工学的な困難さが挙げられます。

詳細な内訳については、[docs/SAFETY_AND_LIMITATIONS.md](./docs/SAFETY_AND_LIMITATIONS.md) を参照してください。
